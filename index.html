Yes, Voidstrap is completely safe to use. It's open-source, so anyone can review the code, and it has been verified by multiple security services. We never collect any personal data or credentials.
No, Voidstrap doesn't modify game files or provide any unfair advantages. It's simply a launcher with quality-of-life improvements. Many popular Roblox content creators use similar tools without issues.
Voidstrap builds upon Bloxstrap with additional features like enhanced performance options, better UI customization, more logging options, and various quality-of-life improvements. We also maintain more frequent updates.
Voidstrap is only officially supported on Windows. Native Mac or Linux support isn't planned for the future.
New Voidstrap releases are released on Github, download Voidstrap.exe and run it to update Voidstrap. We are trying to fix auto-update in an upcoming release.
